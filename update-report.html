Website Update Report

Content Cleanup & Copy Update

This report documents all content changes applied to the Urban Innovations website based on five annotated screenshot PDFs provided by the client. All changes are surgical — no redesign, no new pages, no layout restructuring. The brand, CSS architecture, and all working functionality have been preserved throughout.

Prepared May 2026  ·  5 Pages Updated  ·  17 Changes Applied

Important Note — Fax Number

The fax number (f. 403.288.7174) and contact details (address, phone) visible in the grey contact strip on each page are embedded in PNG background images — not in HTML text. They cannot be removed via HTML editing alone. Zero instances of fax text were found across all HTML files. Removal of the fax from those image assets requires a separate pass using an image editor (e.g. Photoshop, GIMP). This applies to the contact strip visible on: about.html, rent.html, projects.html, and contact.html.
